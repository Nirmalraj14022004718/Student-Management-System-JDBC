# Student Management System (JDBC)

## Project Overview
The Student Management System is a Java-based console application developed using JDBC. 
It allows users to manage student records stored in a MySQL database by performing CRUD operations.

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE
- GitHub

## Features
- Add Student
- View Student
- Update Student
- Delete Student
- Database Connectivity using JDBC

## Database Table
Table Name: student

| Field Name | Data Type |
|------------|-----------|
| id | int |
| name | varchar |
| course | varchar |
| marks | int |

## How to Run the Project
1. Install MySQL and create a database.
2. Create the student table.
3. Open the project in Eclipse IDE.
4. Configure the database connection in the Java code.
5. Run the Java files to perform CRUD operations.
